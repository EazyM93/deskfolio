# 📋 Project Specification

## Product Name

DeskFolio

## Product Type

Native macOS desktop application.

## Product Goal

Help the user manage a personal investment portfolio locally, starting with ETFs and stocks.

## Non-Goals

- DeskFolio is not a web application.
- DeskFolio does not require cloud sync for the MVP.
- DeskFolio does not require broker integration for the MVP.
- DeskFolio does not require automatic market data for the MVP.

## Core Requirements

- Runs locally.
- Stores data in SQLite.
- Supports full database export as a single file.
- Ships as a macOS `.app`.
- Starts with a double click.
- Tracks assets and transactions.
- Supports manually entered historical monthly values.
- Presents a modern JavaFX UI.

## First Supported Asset Types

- ETF
- Stock

## Future Expansion Areas

- Dividends
- Taxes
- Fees
- Multiple portfolios
- Multiple currencies
- Historical market prices
- Import/export workflows
- Backup automation
