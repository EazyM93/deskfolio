# 🔵 DeskFolio 0.1.0 Release Notes

Status: In Progress

## Summary

Version `0.1.0` is planned as the first usable MVP of DeskFolio. Milestones 1 through 4 are implemented; macOS packaging and settings are still pending.

The goal is to deliver a local-first native macOS desktop application that can manage basic assets and transactions.

## ✨ New Features

- Native JavaFX desktop application shell.
- Sidebar navigation foundation.
- Asset management foundation for ETFs and stocks.
- Transaction entry flow.
- New asset creation from the transaction form.
- Transaction-derived dashboard with total invested value, per-asset cards and cumulative monthly chart.

## 🛠 Improvements

- Dark-mode-ready theme foundation.
- Collapsible sidebar with slide transition.
- Local development reset mode through the `local-reset` Maven profile.

## 🗄 Database

Implemented database work:

- SQLite local database.
- Flyway schema migrations.
- Reference data migration.
- Asset and transaction migration.

The planned schema supports:

- Portfolios
- Currencies
- Asset types
- Assets
- Movement types
- Transaction categories
- Portfolio transactions

The schema is designed to leave room for dividends, taxes, fees, multiple portfolios, multiple currencies and historical prices.

Monetary transaction values are stored as integer minor units.

## 🎨 UI

The planned UI includes:

- Dashboard page.
- Transactions page.
- Navigation shell.
- Settings placeholder or foundation.
- Modern styling inspired by IntelliJ IDEA, Apple design, Notion and Raycast.

## 📦 Packaging

- macOS `.app` packaging is still pending.

## ⚠ Known Constraints

- Automatic market data is not planned for this version.
- Broker import is not planned for this version.
- Cloud sync is not planned for this version.
